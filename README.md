#Jogo da Velha em Python

Sobre o Projeto
Este projeto foi desenvolvido como um check point da aula de Python, com o objetivo de aplicar na prática os conceitos de lógica de programação, estruturas condicionais, laços de repetição e funções.

O sistema consiste em um jogo da velha totalmente funcional executado via terminal, permitindo a interação entre dois jogadores.

Funcionalidades

- Laços de repetição (`while`)
- Funções em Python
- Validação de dados
